The IOT Inspector will be ran from the command line, but all the output from the inspector will be reflected on the non-public facing website

The final Dashboard presented will contain features such as:

  • Running a scan on the home network to see which IOT devices are connected
  • The Dashboard will try to use the Mac Address and various scripts to recognise the IOT device
  • User can assign a name their IOT devices
  • User can select to monitor their IOT devices
  • Traffic of the monitored IOT devices will be displayed as graphs
  • Graphs can be exported to be uploaded to the forum on the public facing web app
