Nosy

Our private facing web app (found in the download tab) contains the following features:

  • Running a scan on the home network revealing any devices are connected
  • The Dashboard uses the Mac Address of the devices and various scripts to recognise the IOT device
  • User can select to monitor their IOT devices
  • Traffic of the monitored IOT devices will be displayed in the graphs tab
  • Graphs can be exported to be uploaded to the forum on the public facing web app for users to share their findings or questions.
