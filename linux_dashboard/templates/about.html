Instructions

Discover All

  • This function scans the home network to view all devices connected
  • Scan will take about 1 minute to perform depending on the number of devices connected

Scan IOT Devices

  • This function scans the home network to view all Internet-of-Things (IOT) devices
  • Besides displaying the IP Address of the devices, some additional infomration will also be displayed
  • If scan returns no results, please attempt to scan for the devices again. There are some instances where the scan will provide no results

Scan Network Traffic

  • This is the main function of nosy, which is to view all the network traffic of the IOT devices connected to home network
  • A realtime graph of the bytes of network traffic against which website your IOT device is visiting will be displayed
  • This feature will be run on the Command Line Interface, please proceed to Scan Traffic tab to view in-depth instructions
  • In the current version the graph displayed is not dynamic. So if the traffic volume is too high, graph may be cluttered

Display Network Traffic

  • This displays the graph of a previously scanned network traffic
  • To upload the scanned graph onto "www.nosybackendboyz.com" proceed to static-Images folder located in the folder containing nosy

Ending a session

  • Proceed to End Session tab to end a session and close the website
