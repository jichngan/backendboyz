Instructions for Network Traffic Scanning

As the network traffic scanning will be done using a command line interface, here are some instructions to guide you through the process

  • Do not be afraid of the command line interface, take it as just another application
  • Firstly, navigate to Nosy package you have just downloaded
  • Enter "cd" once first to change to root directory
  • Enter "ls" to list existing folders.
  • Enter "cd Downloads" to proceed to Downloads folder. If you have saved Nosy in another folder navigate using "cd [Folder_Name]" to that folder
  • Proceed to Nosy by entering "cd Nosy"
  • To proceed with the network scan, enter "./sniff.sh"
  • Follow instructions on the command line interface accordingly
